1- Copier le fichier configs.properties et le placer dans un répertoire à la racine de la machine qui contient le serveur (le nom du dossier doit être /isidrone/conf)

exemple :C:\isidrone\conf\configs.properties


2-
Dans Apache NetBeans IDE
Cliquer sur l'onglet Services
cliquer sur la petite flèche à la gauche de Servers
cliquer droit sur Apache Tomcat 
cliquer sur properties
cliquer sur Platform


Donc dans les VM Options, tu dois placer -DISIDRONE_PATH=c:\isidrone\
